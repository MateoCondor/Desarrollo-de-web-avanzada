1.- Abrir una terminal:

Abrir una terminal se puede usar el terminal integrado en Visual Studio Code o la terminal del sistema operativo.
Utiliza el comando cd para navegar al directorio que contiene tu archivo C#. Por ejemplo:

- cd ruta/del/directorio

2.- Compilar y ejecutar el programa:

Utiliza el comando dotnet run para compilar y ejecutar el programa:

-dotnet run

Este comando iniciará la aplicación, y verás el mensaje "Ingrese los datos del empleado:" en la consola. Sigua las instrucciones para ingresar los datos del empleado y del gerente.

3.-Observar los resultados:

Después de ingresar la información, el programa mostrará la información del empleado y del gerente en la consola.

El código también incluye la función MostrarInformacion que utiliza polimorfismo a través de la interfaz para mostrar la información de las entidades (Empleado y Gerente).
